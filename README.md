###  Installation
``Golem`` can be installed with ``pip``:

```
pip install golem-opt
```

### Dependencies
The installation requires:
* ``python >= 3.6``
* ``cython``
* ``numpy``
* ``scipy >= 1.4``
* ``pandas``
* ``scikit-learn``
