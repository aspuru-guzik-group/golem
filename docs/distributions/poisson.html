Poisson¶

class golem.Poisson

Poisson distribution. The lambda parameter will be fitted based on the location of the input sample.

Parameters

  • shift (float (0,1], optional) – Shift parameter between 0 and 1. The rate parameter \(\lambda\) will be determined by the location of the input sample plus this value. If \(x_k\) is the input location and \(\delta\) is the value of this argument, then \(\lambda = x_k + \delta\). Having shift != 0 ensures the distribution has a unique mode.

  • low_bound (float, optional) – Lower bound for the distribution. Default is zero.
