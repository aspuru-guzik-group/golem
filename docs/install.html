Installation¶

Golem can be installed with pip:

pip install matter-golem

Alternatively, you can clone the GitHub repo and install it from source:

git clone https://github.com/aspuru-guzik-group/golem.git
cd golem
pip install .

Requirements¶

  • python >= 3.7

  • numpy

  • scipy >= 1.4

  • pandas

  • scikit-learn
